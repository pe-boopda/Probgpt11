## Типы тестов (планируемые)

1. **Множественный выбор** - классические тесты с вариантами ответов
2. **Таблица соответствий** - сопоставление элементов
3. **Аннотация изображений** - рисование и подписи на фото
4. **Свободный ответ** - текстовое поле
5. **Последовательность** - упорядочивание элементов
6. **Hotspot** - выбор области на изображении
7. **Fill in the blanks** - заполнение пропусков

## База данных (основные таблицы)

- **users** - пользователи (студенты, преподаватели, админы)
- **groups** - учебные группы
- **tests** - тесты
- **questions** - вопросы
- **question_options** - варианты ответов
- **test_sessions** - сессии прохождения тестов
- **answers** - ответы студентов
- **results** - результаты тестирования
- **images** - изображения для тестов
- **annotations** - аннотации на изображениях

## Технологический стек

### Backend
- FastAPI
- SQLAlchemy
- Alembic (миграции)
- Pydantic
- python-jose (JWT)
- passlib (хеширование паролей)
- Pillow (обработка изображений)
- pandas (статистика)

### Frontend
- React 18
- Vite
- React Router
- Axios
- Fabric.js (рисование на canvas)
- Chart.js (графики)
- TailwindCSS
- Zustand или Redux (state management)

### База данных
- PostgreSQL

### Дополнительно
- Docker & Docker Compose
- Nginx (для production)

## Этапы разработки

### Этап 1: Базовая настройка ✓
1. Создание структуры проекта
2. Настройка FastAPI
3. Настройка React + Vite
4. Подключение PostgreSQL

### Этап 2: Аутентификация
1. Модели пользователей
2. JWT токены
3. Регистрация/Вход
4. Роли (студент, преподаватель, админ)

### Этап 3: Основной функционал
1. CRUD для тестов
2. Типы вопросов
3. Прохождение тестов
4. Сохранение результатов

### Этап 4: Работа с изображениями
1. Загрузка изображений
2. Canvas для рисования
3. Сохранение аннотаций
4. Проверка ответов с изображениями

### Этап 5: Админ-панель
1. Создание тестов
2. Управление группами
3. Просмотр результатов
4. Статистика и аналитика

### Этап 6: AI функционал
1. Автопроверка текстовых ответов
2. Анализ изображений
3. Рекомендации

### Этап 7: Деплой
1. Docker контейнеризация
2. Настройка для локальной сети
3. Документация
